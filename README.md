# Sandesh Mane — Portfolio

React + TypeScript + Tailwind CSS + Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open the URL it prints (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

## Deploy (Vercel — recommended, matches your other projects)

1. Push this folder to a new GitHub repo.
2. Go to [vercel.com](https://vercel.com), "Add New Project", import the repo.
3. Framework preset: **Vite**. Leave build settings as default. Deploy.

## Notes

- Your resume is at `public/Sandesh_Mane_Resume.pdf` — replace this file (keep the same
  filename) any time you update your resume, no code changes needed.
- Update project cards, skills, and certifications directly in `src/components/`.
- Colors and fonts are defined as design tokens in `tailwind.config.js`.
